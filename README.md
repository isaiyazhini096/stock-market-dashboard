# 📈 Power BI Sales Dashboard

## Project Overview
This project presents an interactive Sales Dashboard built using Power BI. The dashboard provides insights into sales performance across different regions, products, and categories through interactive visualizations.

## Features
- KPI Cards for key metrics
- Sales Trend Analysis
- Revenue by Region
- Product-wise Revenue Analysis
- Interactive Slicers and Filters
- Pie Charts and Bar Charts
- Data-driven Insights

## Tools Used
- Power BI
- Microsoft Excel

## Dataset
The dashboard uses the Global Superstore dataset for sales analysis.
